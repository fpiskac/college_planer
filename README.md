Naziv projekta: Praćenje bodova po predmetima

Opis:
Single page web aplikacija izrađena u Vue 3.
Aplikacija omogućuje dodavanje predmeta, zadataka i praćenje ostvarenih bodova.

Pokretanje projekta:
1. Otvoriti terminal u root direktoriju projekta
2. Pokrenuti: npm install
3. Pokrenuti: npm run dev
4. Aplikacija je dostupna na adresi koju Vite ispiše u terminalu

Napomena:
Podaci se spremaju u localStorage preglednika.
